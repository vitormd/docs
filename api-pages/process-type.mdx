---
title: "Tipos de Processo"
description: "Lista dos tipos de processos disponíveis no onboarding e suas descrições."
---

| **Enum**     | **Descrição**                       |
| ------------ | ----------------------------------- |
| `FRAUD`      | Processo de verificação de fraude  |
| `LIVENESS`   | Processo de detecção de vivacidade |
| `VALIDATION` | Processo de validação de dados     |
| `FACEMATCH`  | Processo de comparação facial — oculto no Swagger |

> ℹ️ Este enum é usado para identificar os diferentes tipos de verificação no fluxo de onboarding.
