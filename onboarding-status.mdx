---
title: "Status do cadastro"
description: "Lista dos possíveis status do processo de onboarding e suas descrições."
---

| **Enum**      | **Descrição**                              |
| ------------- | ------------------------------------------ |
| `STARTED`     | Cadastro iniciado                          |
| `PENDING`     | Cadastro pendente (aguardando documentos)  |
| `PROCESSING`  | Cadastro em processamento                  |
| `ANALYSIS`    | Cadastro em análise (Backoffice)           |
| `DONE`        | Cadastro finalizado com sucesso            |
| `REJECTED`    | Cadastro rejeitado                         |
| `CANCELED`    | Cadastro cancelado                         |
| `ERROR`       | Cadastro com erro (falha no processamento) |
| `BLACKLISTED` | Cadastro em blacklist                      |
| `RESOLVED`    | Cadastro com pendências resolvidas         |

> ℹ️ Este enum é utilizado em diversos pontos da API, especialmente nas respostas de status de endpoints relacionados a onboarding.
