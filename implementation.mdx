## API

Através de nossas APIs é possível fazer o uso completo de nosso sistema todas nossas propostas de valor

- Coleta de biometria e documentos
- Prova de vida - Liveness
- Comparação de faces - Facematch
- Checagem de dados briográficos - Background Check
- Validação de identidade completa - Cadastro (Onboarding)

Para a necessidade de coleta direto em seu sistema de maneira integrada, verifique nossa [API de Step Services](/api-reference/steps-services/inicia-o-processo-de-liveness). Através deles é possível gerar URLs únicas para executar a coleta e continuar o fluxo do usuário dentro da sua jornada.
