**Formatos de oferta**

A solução de onboarding digital pode ser oferecida nos seguintes formatos:

- APIs
  - API services - serviços de antifraude
  - API steps - criação de URLs por etapa de onboarding
- App/Dash
- SDKs
  - Liveness e Facematch
  - Captura Documental

### Serviços

O onboarding pode ser composto por diversos serviços, que habilitam a verificação de identidade, sendo eles:

- Autenticação
- Cadastros
  - Realizar análise de cadastro
- Serviços biométricos
  - <Tooltip tip="Realiza uma análise automática para verificar se a imagem enviada pertence a uma pessoa real e está viva, evitando tentativas de fraude como fotos, vídeos ou máscaras. O processo ocorre sem necessidade de interação ativa do usuário.">Liveness</Tooltip>
  - Facematch
- Análise de fraude
  - CPF e Documento
  - CPF, Documento e Face
  - Documentoscopia (Análise de fraude documental)
- Serviços de Background Check
- OCR de documento padronizado
