  <Card title="Liveness" icon="face-viewfinder" iconType="sharp-solid" horizontal cta="Realiza uma análise automática para verificar se a imagem enviada pertence a uma pessoa real e está viva, evitando tentativas de fraude como fotos, vídeos ou máscaras.     Realiza uma análise automática para verificar se a imagem enviada pertence a uma pessoa real e está viva, evitando tentativas de fraude como fotos, vídeos ou máscaras. O processo ocorre sem necessidade de interação ativa do usuário.
  </Card>
  <Card title="Facematch" icon="people-arrows" iconType="sharp-solid" horizontal>
    Compara imagens faciais para verificar similaridade e confirmar a identidade do usuário. Pode operar no modo **1:1** (comparação entre dois rostos) ou **1:N** (comparação contra um banco de imagens), permitindo autenticação segura, prevenção de fraudes e verificação eficiente em diferentes contextos.
  </Card>
</CardGroup>

## Formatos de oferta

A solução de onboarding digital pode ser oferecida nos seguintes formatos:

- APIs
  - API services - serviços de antifraude
  - API steps - criação de URLs por etapa de onboarding
- App/Dash
- SDKs
  - Liveness e Facematch
  - Captura Documental
